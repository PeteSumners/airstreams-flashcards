## Quick Start (3 Steps)

### 1. Install Anki (if not already installed)
   - Double-click: **Install_Anki.bat**
   - OR visit: https://apps.ankiweb.net/ and download the Windows version

### 2. Launch Anki
   - Double-click: **Launch_Anki.bat**
   - OR start Anki from your Start Menu

### 3. Import Your Flashcards
   In Anki:
   - Click "File" ‚Üí "Import..."
   - Select: **OSHA_Flashcards_Anki.txt**
   - Set "Fields separated by" to: **Tab**
   - Click "Import"
   - Start studying!

- **Free & Open Source** - No cost, community-driven
- **Spaced Repetition** - Scientifically proven to improve memory
- **Mobile Apps** - Study on phone, tablet, or computer
- **Sync Across Devices** - Free cloud sync with AnkiWeb
- **Works Offline** - No internet required for studying
