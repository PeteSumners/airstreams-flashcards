================================================================================
TROUBLESHOOTING
================================================================================

**GitHub Pages not loading?**
- Give it 2-3 minutes after first push
- Check https://github.com/PeteSumners/training-video-flashcards/settings/pages
- Make sure Source is set to "master branch /docs folder"

**Anki won't install?**
- Right-click anki-installer.exe ‚Üí Run as Administrator
- Check C:\Program Files\Anki\ after installation

**Quizlet import failed?**
- Make sure delimiter is set to TAB (not comma)
- Copy directly from OSHA_Flashcards_Quizlet.txt
